Este repositorio contiene los datos y los análisis sobre biodiversidad de lepidopteros en Sierra Nevada (S Spain). Se trata de los análisis realizados para preparar el manuscrito titulado: Butterfly diversity, richness and abundance patterns in Sierra Nevada (SE Spain): conservation implications under a global change scenario.

Estructura del repositorio

  • Prepara los datos de diversidad, abundancia y riqueza.

  • Variación de riqueza, diversidad y abundancia con la elevación

  • Tabla de Descriptivos

  • Prepara la matrix de datos ambientales

  • Modelos

  • Gráficos descriptores modelos

Otras cosas

  • Petición de JMBA sobre datos de diversidad, abundancia y riqueza de todo el periodo. También se pueden ver los plots asociados a esos datos (e.g. Variación de riqueza, diversidad y abundancia con la elevación) aquí
